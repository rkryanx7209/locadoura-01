# 🚗 Sistema de Locadora 2.0

Uma plataforma moderna e responsiva para **gestão de aluguel de veículos**, desenvolvida com foco em **experiência do usuário, organização da interface e performance**.

O sistema permite visualizar veículos disponíveis, realizar buscas e aplicar filtros de forma rápida e intuitiva.

---

## 🌐 Acesse o sistema

🔗 https://locadoura-ryan01.netlify.app/

---

## ⚙️ Funcionalidades

- 🚗 Interface intuitiva e responsiva  
- 📋 Listagem dinâmica de veículos  
- 🔎 Sistema de busca e filtros  
- 📱 Layout otimizado para dispositivos móveis  
- ⚡ Navegação rápida e leve

---

## 🛠 Tecnologias Utilizadas

- **Frontend:** HTML5 estruturado e CSS3 moderno  
- **Lógica:** JavaScript (manipulação de DOM e eventos)  
- **Hospedagem:** Netlify (deploy e CI/CD)

---

## 🎯 Objetivo do Projeto

Este projeto foi desenvolvido para praticar e demonstrar habilidades em:

- desenvolvimento web frontend  
- criação de interfaces responsivas  
- manipulação de elementos com JavaScript  
- publicação de aplicações na web
